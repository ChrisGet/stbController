Command Sequence Control

Command Sequences allow you to send multiple button press commands to STBs at the click of a single button

Any sequences you have already created will be shown to the right. Hover your mouse over the title to see at a glance what commands that sequences contains

Import "Stress" script (.txt file format only)

These usually come from Brentwood and are used in the stress tests

NOTE: Due to the differences between our sequences and the stress scripts, the conversion will be a "best guess" attempt. If a stress script includes mutliple STBs, multiple sequences will be created with the appropriate STB name attached.
